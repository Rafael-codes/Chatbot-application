# Local Chatbot using GPT4All (Socket + Tkinter)

This project is a **local AI chatbot application** built using **Python**, **GPT4All**, **socket programming**, and **Tkinter GUI**.  
The chatbot runs **fully offline** using a locally stored large language model and communicates through a client–server architecture.

---

##  Features

- Fully offline AI chatbot
- Uses GPT4All local LLM (Llama-3.2-3B-Instruct)
- TCP socket-based client–server communication
- Tkinter-based graphical user interface
- One-line AI responses for fast interaction
- Error handling for model loading and server connection
